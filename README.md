# 💰 defi-yield-aggregator

Yield farming aggregator that auto-optimizes across protocols

## Features

- Multi-protocol yield farming
- Auto-rebalancing
- Strategy vaults
- Fee management
- APY optimization

## Tech Stack

- **Smart Contracts:** Solidity ^0.8.20
- **Framework:** Hardhat
- **Frontend:** Next.js + TypeScript
- **Web3:** ethers.js / wagmi

## Getting Started

```bash
npm install
npx hardhat compile
npx hardhat test
npx hardhat run scripts/deploy.js --network localhost
```
